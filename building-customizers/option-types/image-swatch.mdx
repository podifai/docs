The **Image Swatch** option type allows your customers to select from a visual gallery of images. This is perfect for showcasing different styles, patterns, or designs.

## When to Use Image Swatch

Use the Image Swatch option type when you want to:

-   Display different product styles or patterns visually
-   Show material textures (e.g., wood grain, fabric patterns)
-   Let customers choose from design variations

## How to Add an Image Swatch

1.  In the option set editor, click **Add Option**.
2.  Select **Image Swatch** from the list of option types.
3.  Give your option a name (e.g., "Pattern Style").
4.  Upload images for each swatch option.
5.  Optionally, add a label and price adjustment for each image.

![Adding an Image Swatch option](/images/screenshots/option-types/image-swatch-setup.png)
*Upload images and configure each swatch option.*

## Configuration Options

### Image Upload

Upload high-quality images that clearly represent each option. Recommended image size is 200x200 pixels.

### Labels

Add descriptive labels to each image swatch (e.g., "Floral Pattern", "Geometric Design").

### Pricing

You can add an extra charge for specific image options. For example, premium designs could cost an additional $5.

## Example Use Cases

-   **Jewelry**: Different gemstone cuts or settings
-   **Apparel**: Fabric patterns or prints
-   **Home Goods**: Wood finishes or upholstery patterns

## Best Practices

<Tip>
Use consistent image dimensions for all swatches to ensure a clean, professional appearance.
</Tip>

<Warning>
Avoid using images that are too large, as this can slow down your product page loading time.
</Warning>
