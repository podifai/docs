The **Color Swatch** option type displays a palette of colors for your customers to choose from. This is ideal for products that come in multiple color variations.

## When to Use Color Swatch

Use the Color Swatch option type when you want to:

-   Offer different color options for your product
-   Display metal finishes (e.g., Gold, Silver, Rose Gold)
-   Show paint or fabric colors

## How to Add a Color Swatch

1.  In the option set editor, click **Add Option**.
2.  Select **Color Swatch** from the list of option types.
3.  Give your option a name (e.g., "Metal Finish").
4.  Add color values using hex codes (e.g., #FFD700 for gold).
5.  Optionally, add a label and price adjustment for each color.

## Configuration Options

### Color Values

Enter the hex code for each color you want to offer. You can use a color picker tool to find the exact hex code.

### Labels

Add descriptive names for each color (e.g., "Midnight Black", "Ocean Blue").

### Pricing

You can add an extra charge for premium colors or finishes.

## Example Use Cases

-   **Jewelry**: Metal finishes (Gold, Silver, Platinum)
-   **Apparel**: Fabric colors
-   **Home Decor**: Paint colors or finishes

## Best Practices

<Tip>
Use clear, descriptive labels for your colors to help customers make informed decisions.
</Tip>

<Note>
Consider offering a limited number of colors (5-10) to avoid overwhelming your customers with too many choices.
</Note>
