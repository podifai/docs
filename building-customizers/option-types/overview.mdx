## Introduction

Podifai offers **9 powerful option types** that cover every product customization scenario. Each option type is designed for specific use cases and can be displayed in either **Standard** or **Collapsible (Accordion)** mode.

## Display Modes

Every option type supports two display modes:

**Standard Mode**: All options visible at once, best for simple quick selections

**Collapsible Mode**: Options in collapsible accordion sections, great for mobile and guided customization

<Tip>
You can mix and match display modes. For example, use Standard mode for "Size" and Collapsible mode for "Add Engraving".
</Tip>
