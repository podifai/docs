Welcome to the Podifai Quick Start Guide! This guide will walk you through the essential steps to get your first product customizer live on your Shopify store.

## Step 1: Install Podifai

First, you need to install the Podifai app from the Shopify App Store.

1.  Go to the [Podifai App Store page](https://apps.shopify.com/podifai).
2.  Click the **Install** button.
3.  Follow the on-screen instructions to approve the installation.

![Install Podifai from the Shopify App Store](/images/screenshots/quick-start/install-app.png)
*Click the "Install" button on the Shopify App Store.*

## Step 2: Create an Option Set

Next, you'll create an option set. An option set is a group of custom options that you can apply to your products.

1.  From your Podifai dashboard, click on **Option Sets**.
2.  Click the **Create Option Set** button.
3.  Give your option set a name (e.g., "Jewelry Engraving").
4.  Add your first option, such as a **Text Input** for the engraving text.

![Create a new Option Set in Podifai](/images/screenshots/quick-start/create-option-set.png)
*The Option Set editor allows you to add and configure various option types.*

## Step 3: Assign to a Product and Publish

Finally, assign your newly created option set to a product and publish it to your store.

1.  In the Option Set editor, go to the **Product Assignment** tab.
2.  Select the product you want to add the customizer to.
3.  Click **Save**.
4.  Click the **Publish** button to make your customizer live.

![Assign the Option Set to a product and publish](/images/screenshots/quick-start/assign-and-publish.png)
*Your product customizer is now live on your Shopify store!*
