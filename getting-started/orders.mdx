The **Orders** page displays all orders that include products with Podifai customizations. Use this page to review customer selections, fulfill orders, and track customized products.

## Accessing Orders

Click **Orders** in the left navigation menu to view your customized orders.

## Order List

The Orders page displays a table with the following information for each order:

| Column | Description |
|--------|-------------|
| **Order ID** | The Shopify order number (e.g., #1001) |
| **Product** | Names of all customized products in the order |
| **Payment status** | Payment status (Paid, Pending, etc.) |
| **Fulfillment status** | Fulfillment status (Unfulfilled, Fulfilled, etc.) |
| **Total** | Total order amount |
| **Date created** | When the order was placed |

![Orders list](/images/screenshots/orders/orders-list.png)

## Searching Orders

Use the search bar at the top right to find specific orders:

1.  Enter a product title in the search field.
2.  Results filter automatically as you type.
3.  Clear the search field to see all orders again.

## Viewing Order Details

Click on any order row to open the order details drawer:

### Order Information

-   **Order ID**: The Shopify order number
-   **Payment Status**: Current payment status
-   **Fulfillment Status**: Current fulfillment status

### Product Details

For each customized product in the order, you'll see:

-   **Product Name**: Clickable link to the product in Shopify admin
-   **Price**: Product price
-   **Customization Options**: All customer selections displayed as name-value pairs

![Order details drawer](/images/screenshots/orders/order-details.png)
*The order details drawer shows all customization choices for each product.*

## Understanding Customization Data

Each customization option shows:

-   **Option Name**: The label you configured (e.g., "Metal Finish", "Engraving Text")
-   **Selected Value**: What the customer chose or entered

For example:
-   Metal Finish: Gold
-   Engraving Text: "Happy Birthday Mom"
-   Number of Pendants: 3

## Pagination

If you have many orders, use the **Previous** and **Next** buttons at the bottom of the table to navigate between pages.

<Note>
Only orders containing products with Podifai customizations appear on this page. Regular orders without customizations are managed directly in Shopify.
</Note>

## Fulfilling Customized Orders

When fulfilling orders with customizations:

1.  Review all customization details in the order drawer.
2.  Use the customization values for production (engraving, color selection, etc.).
3.  Fulfill the order through Shopify as usual.

<Tip>
Click the product name link in the order drawer to quickly access the product in your Shopify admin.
</Tip>
