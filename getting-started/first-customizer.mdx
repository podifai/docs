This guide will walk you through creating your first product customizer from scratch. By the end of this tutorial, you'll have a fully functional customizer live on your Shopify store.

## What You'll Build

In this tutorial, you'll create a simple jewelry engraving customizer that allows customers to:

-   Choose a metal finish (e.g., Gold, Silver, Rose Gold)
-   Enter custom engraving text
-   Select the number of discs

## Prerequisites

Before you begin, make sure you have:

-   Installed the Podifai app on your Shopify store
-   At least one product in your Shopify store that you want to add the customizer to

## Step 1: Create a New Option Set

1.  From your Podifai dashboard, navigate to **Option Sets**.
2.  Click the **Create Option Set** button.
3.  Name your option set "Jewelry Engraving".

![Create a new option set](/images/screenshots/first-customizer/create-option-set.png)
*Give your option set a descriptive name.*

## Step 2: Add a Color Swatch Option

1.  In the option set editor, click **Add Option**.
2.  Select **Color Swatch** from the list of option types.
3.  Name the option "Metal Finish".
4.  Add three color values: Gold (#FFD700), Silver (#C0C0C0), and Rose Gold (#B76E79).

![Add a color swatch option](/images/screenshots/first-customizer/add-color-swatch.png)
*Configure the color swatch with your desired colors.*

## Step 3: Add a Text Input Option

1.  Click **Add Option** again.
2.  Select **Text Input** from the list.
3.  Name the option "Engraving Text".
4.  Set a character limit of 20 characters.

![Add a text input option](/images/screenshots/first-customizer/add-text-input.png)
*Set a character limit to control the length of the engraving.*

## Step 4: Add a Dropdown Option

1.  Click **Add Option** one more time.
2.  Select **Dropdown** from the list.
3.  Name the option "Number of Discs".
4.  Add three values: "One", "Two", "Three".

![Add a dropdown option](/images/screenshots/first-customizer/add-dropdown.png)
*A dropdown is perfect for a limited set of choices.*

## Step 5: Assign to a Product

1.  Go to the **Product Assignment** tab in the option set editor.
2.  Search for the product you want to add the customizer to.
3.  Select the product from the list.
4.  Click **Save**.

![Assign the option set to a product](/images/screenshots/first-customizer/assign-product.png)
*Select the product you want to customize.*

## Step 6: Preview and Publish

1.  Click the **Preview** button to see how your customizer looks on the product page.
2.  If everything looks good, click the **Publish** button to make it live.

![Preview and publish your customizer](/images/screenshots/first-customizer/preview-publish.png)
*Preview your customizer before publishing.*

Congratulations! You've just created your first product customizer with Podifai. Visit your product page to see it in action.
